[FOSBlog](/) adalah sebuah open-source blogging untuk para content creator terlebih untuk para developer yang sedang mempelajari open-source dan git. Mereka dapat dengan bebas membuat artikel apapun tentang dunia IT selama tidak melanggar [Code of Conduct](/code-of-conduct) website ini. Orang-orang juga dapat menjadikan repo ini untuk event _Hacktoberfest_ yang biasanya diselenggarakan tiap tahun di bulan Oktober.

## Arti dari "FOSBlog"

FOSBlog sendiri adalah singkatan dari **Free Open-source Blog**, dimana orang-orang dapat membuat dan mempublikasikan artikelnya dengan cara membuat [Pull Request](https://github.com/UnboxLabs/fosblog/pulls) baru. Untuk lebih detail mengenai cara pembuatan post baru bisa cek [disini](/create-post).

## Format Post/Artikel

Pada platform FOSBlog, mengadopsi format artikel menggunakan Markdown. Format ini lebih aman digunakan dibanding penulisan menggunakan HTML dikarenakan dapat menghindari js script inject. Markdown juga bisa terlihat lebih simple penulisannya dibandingkan HTML code.

Jadi, tunggu apalagi ? Langsung cobain yuk berkontribusi di open-source blog
